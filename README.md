# Retail Store Intelligence Challenge

## Overview

This project is a menu-driven C++ application developed as part of the Retail Store Intelligence Challenge. It simulates common retail business operations and demonstrates how software can be used to automate billing, taxation, discount management, profitability analysis, and performance reporting.

The application consolidates five retail-focused modules into a single executable, providing a structured approach to solving real-world business problems using fundamental programming concepts.

---

## Features

### Smart Billing Assistant

Calculates the total cost of a purchase based on:

* Product Name
* Unit Price
* Quantity

### Discount Engine

Computes:

* Discount Amount
* Final Payable Price

based on the product price and discount percentage.

### GST Billing System

Generates GST-inclusive billing information by calculating:

* GST Amount
* Final Invoice Value

### Retail Profit Analyzer

Performs profitability analysis using:

* Cost Price
* Selling Price
* Units Sold

and calculates:

* Investment
* Revenue
* Profit
* Profit Percentage

### Daily Performance Dashboard

Generates key retail performance metrics:

* Gross Sales
* Discount Amount
* Net Sales
* Operating Profit
* Profit Margin
